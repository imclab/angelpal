angelpal
========

HOW TO INSTALL
============

install nodejs, npm, mysql-server

DATABASE
========
Create database : angelpal

SERVER
======
/server$ npm install
<br>
start server : /server$ node app.js

CLIENT
======
$ npm install -g bower
<br>
$ npm install -g grunt
<br>
$ npm install -g grunt-cli
<br>
/client$ npm install | bower install
<br>
start client : /client$ grunt serve
